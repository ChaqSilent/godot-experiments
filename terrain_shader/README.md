# Terrain shader

![Screenshot](./screenshot.png?raw=true)

A simple spatial shader that mixes 2 textures depending on the terrain slope.

Inspired by [this tutorial](http://ffxrunner2.ffx.it/simple-terrain-shader-tutorial/).
